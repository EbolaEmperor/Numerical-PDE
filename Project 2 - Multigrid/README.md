## June 20, 2023

对于2维边值问题，添加了9点离散格式，可以达到4阶收敛。要求用户提供$\Delta f$，调用方法参考`examples/test9stencil-64.json`。

性能上提升巨大，使用传统5点离散格式，当$n=1024$时，运算时间为3.976s，精度只有`3.37e-08`；使用9点离散格式，设$n=64$，运算时间为0.054s，且精度达到`4.16e-10`，进一步增大$n$可以获得更高精度。
